# Login Page

Este projeto foi desenvolvido com o objetivo exclusivo de estudo e testes. Ele serve como um ambiente experimental para validar conceitos fundamentais de back-end, como gerenciamento de tokens, sistema de login, autenticação e autorização, codificação e hash de senhas, além de criptografia de dados sensíveis.

Não se trata de um sistema em produção, mas sim de um laboratório prático para entender o funcionamento interno desses mecanismos, testar fluxos de segurança e boas práticas de desenvolvimento.

## Técnologias:
 * Node.js
 * React.js
 * Mongodb
